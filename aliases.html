Robot folders aliases¶

robot_folders defines a number of bash aliases that help you typing even less. They aren’t necessarily as easy and logical to remember as for example fzirob change_environment, but once you get to know them, you’ll probably enjoy simply typing ce instead.

Alias

fzirob command

ce

fzirob change_environment

cdr

fzirob cd

cdros

fzirob cd ros

cdcol

fzirob cd colcon

cdmisc

fzirob cd misc

cdhome

fzirob cd

fm

fzirob make

fmc

fzirob make colcon

fmcp

fzirob make colcon –colcon-args –packages-select
